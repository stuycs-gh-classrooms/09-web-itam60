Koch Curve

Basic curve:

draw_koch(t,4,25)

This is the basic Koch curve. At depth 2, it consists of a straight line, two lines that meet at a 60-degree angle, and another straight line.

My modified Koch curve. It consists of 4 segments instead of 5, and segments meet at 90-degree angles instead of 60-degree ones. The width also varies randomly with depth, and the size decreases as the depth increases.

Sierpinski Triangle

Basic triangle:

sierpinski(t,4,30)

This is the basic Sierpinski triangle. At depth 2, it consists of 3 equilateral triangles that form a larger equilateral triangle together.

Modified triangle:

    
      def triangle2(t,size):
        t.lt(72)
        t.fd(size)
        t.rt(144)
        t.fd(size)
        t.rt(108)
        t.fd(size*0.618)
        t.rt(180)

      def sierpinski_2(t,depth,length):
        if depth==1:
          triangle2(t,length)
        elif depth>1:
          t.pencolor(255,0,0)
          sierpinski_2(t,depth-1,length/2)
          t.fd(length*0.5*0.618)
          t.pencolor(0,255,0)
          sierpinski_2(t,depth-1,length/2)
          t.bk(length*0.5*0.618)
          t.lt(72)
          t.fd(length/2)
          t.rt(72)
          t.pencolor(0,0,255)
          sierpinski_2(t,depth-1,length/2)
          t.rt(108)
          t.pencolor(255,0,0)
          t.fd(length/2)
          t.lt(108)

        def sierpinski_star(t,depth,length):
          i=0
          while i<5:
            sierpinski_2(t,depth,length)
            t.fd(length*0.618)
            t.rt(72)
            i+=1
sierpinski_star(t,4,100)

My modified Sierpinski triangle. The base is an isosceles triangle with base angles of 72 degrees instead of an equilateral triangle, and for depths greater than 1 the constituent triangles are red, green, and blue. My program actually draws 5 Sierpinski triangles of the given depth at 108 degrees to each other, creating a five-pointed star.

Fractal Tree

Basic tree:

tree(t,5,50,20)

This is a basic fractal tree. At depth 2, it consists of a straight line from which 2 other "branches" diverge, each at an angle to the fist line.

Modified tree:

    
      def tree_2(t,depth,length,angle):
      if depth==1:
          t.fd(length)
          t.bk(length)
      elif depth>1:
          color=random_color()
          t.pencolor(color)
          t.fd(length)
          tree_2(t,depth-1,length*0.5,angle)
          t.rt(angle)
          tree_2(t,depth-1,length*0.5,angle)
          t.rt(angle)
          tree_2(t,depth-1,length*0.5,angle)
          t.lt(3*angle)
          tree_2(t,depth-1,length*0.5,angle)
          t.lt(angle)
          tree_2(t,depth-1,length*0.5,angle)
          t.rt(2*angle)
          t.bk(length)
tree_2(t,6,100,60)

My modified fractal tree. Each line diverges into 5 branches instead of 2, the length of the branches decreases as depth increases, and the color varies randomly with depth.
